# Lección 1: Introducción a HTML

## ¿Qué es HTML?

HTML significa Lenguaje de Marcado de Hipertexto (HyperText Markup Language). Es el lenguaje de marcado estándar para documentos diseñados para mostrarse en un navegador web.

## Estructura básica de HTML

Cada documento HTML tiene una estructura básica:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página web</title>
  </head>
  <body>
    <h1>¡Hola, mundo!</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>
```

## Elementos comunes de HTML

- `<h1>` a `<h6>`: Encabezados
- `<p>`: Párrafo
- `<a>`: Enlace (ancla)
- `<img>`: Imagen
- `<ul>` y `<li>`: Lista desordenada y elementos de lista

## Tu primer proyecto

Creemos una página de blog sencilla sobre tu hobby favorito. Por ejemplo, si te gusta el fútbol, podrías escribir sobre el último partido que viste.

### Pasos

1. Abre un editor de texto (como VS Code).
2. Crea un archivo nuevo llamado `index.html`.
3. Copia la estructura básica de HTML anterior.
4. Cambia el título a "Mi Blog de Fútbol".
5. Añade un encabezado con el nombre de tu blog.
6. Escribe un párrafo describiendo tu equipo favorito.

## Ejemplo: Mi primer blog

Aquí tienes un ejemplo de cómo podría verse tu HTML:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi Blog de Fútbol</title>
  </head>
  <body>
    <h1>¡Bienvenido a Mi Blog de Fútbol!</h1>
    <p>Hoy vi un partido increíble entre el Barcelona y el Real Madrid.</p>
    <p>El marcador final fue 2-1. ¡Qué partidazo!</p>
  </body>
</html>
```
